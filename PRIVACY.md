# Privacy Policy — SynapLink

_Last updated: June 30, 2026_

SynapLink is a macOS application that does not collect, store, transmit, or share any user data.

## Data collection

SynapLink does not collect any personal data, usage data, or analytics. The app:

- Operates entirely locally on your Mac
- Has no network connectivity and does not communicate with any server
- Does not use third-party SDKs, trackers, or analytics frameworks
- Does not require an account, sign-in, or registration

## File access

SynapLink runs inside the macOS App Sandbox. It only accesses folders that you explicitly select via the native macOS file picker (`NSOpenPanel`). All file operations (creating symbolic links or copies) occur exclusively within those user-selected locations. No file content ever leaves your Mac.

## Changes to this policy

If this policy changes in the future, the updated version will be published at this same URL.
