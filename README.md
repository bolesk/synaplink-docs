SynapLink is a macOS app for developers who work with AI coding agents — Claude Code, Cursor, Windsurf, and others. It lets you keep all your agent configuration files (prompts, rules, MCP definitions, sub-agent instructions) in a single central library, and distribute them to your projects as symbolic links or copies with one click.

## The problem

Every AI tool expects its config files in a different location and format. When you switch tools — or work across multiple projects — you end up manually copying and updating the same files everywhere. Configurations drift, get out of sync, and become a maintenance burden.

## The solution

SynapLink gives you a **single source of truth**: one library folder that feeds all your projects, all your tools, all at once. Update a file in the library and every linked project reflects the change instantly.

- **Link mode** — creates a symbolic link in the project pointing to the library. Changes propagate automatically.
- **Copy mode** — creates an independent copy, useful for files that need project-specific customization.

## Why provider-independent

AI providers are building their own config-sharing ecosystems — but they have no incentive to make migration to competitors easy. SynapLink sits outside that ecosystem: a local, provider-agnostic layer that works regardless of which AI agent you're using today or switching to tomorrow.

## Requirements

- macOS 15.0 (Sequoia) or later
- Apple Silicon or Intel Mac
